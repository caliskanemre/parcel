# Webflux soap app

## Prerequisites
* Install Java
* Install modules: load gradle project
* Create new mock service from soap-ui as importing wsdl and run it on port 8088
* run app
* send post message over postman or soapui
* it is working on port 9090
* sample message is 
* {
  "id": 5,
  "message" : "message1"
  }
